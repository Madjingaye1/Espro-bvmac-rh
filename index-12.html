🔥 Configuration Firebase

Pour synchroniser les données entre tous les utilisateurs, configurez votre base de données Firebase. C'est gratuit et prend 5 minutes.

1
Créer un projet Firebase
Allez sur console.firebase.google.com → Créer un projet → nommez-le bvmac-rh
2
Activer Realtime Database
Menu gauche → Realtime Database → Créer une base → Mode test → Activer
3
Récupérer la configuration
⚙️ Paramètres du projet → Vos applications → icône </> → Enregistrer → Copier firebaseConfig
4
Configurer les règles (temporaire)
Realtime Database → Règles → Mettre ".read": true, ".write": true → Publier
